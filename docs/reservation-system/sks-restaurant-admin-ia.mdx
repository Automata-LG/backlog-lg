## 4. 좌석 관리

### 4.1 좌석 현황

<Card title="실시간 좌석 현황" icon="table-cells">
  **좌석 구성**
  - 룸: 1개 (최대 8인)
  - 홀: 8개 테이블
    - 4인 테이블: H4-1 ~ H4-6 (6개)
    - 6인 테이블: H6-1 ~ H6-2 (2개)
  
  **상태 표시**
  - 예약 가능 (초록색)
  - 예약됨 (노란색)
  - 이용 중 (빨간색)
  - 정비 중 (회색)
</Card>

### 4.2 좌석 배정

<Tabs>
  <Tab title="자동 배정">
    **배정 규칙**
    - 인원에 맞는 최적 테이블 자동 선택
    - 4인 이하 → 4인 테이블 우선
    - 5-6인 → 6인 테이블 배정
    - 룸 요청 시 자동 배정
  </Tab>

  <Tab title="수동 배정">
    **수동 선택**
    - 테이블 맵에서 직접 선택
    - 드래그 앤 드롭으로 재배정
    - 특별 요청사항 고려한 배정
  </Tab>
</Tabs>

### 4.3 좌석 설정

<Card title="좌석 설정 관리" icon="sliders">
  **테이블별 설정**
  - 좌석 번호/이름
  - 최소/최대 인원
  - 운영 상태 (활성/비활성)
  - 특별 속성 (창가, 프라이빗 등)
  
  **일괄 설정**
  - 시간대별 운영 설정
  - 특정 날짜 블록 설정
  - 정비 일정 관리
</Card>
