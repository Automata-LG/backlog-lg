## 1. 네비게이션 구조

```
├── 아트센터
│   ├── 호스트 관리
│   ├── 컨텐츠 관리
│   └── 예약 관리
└── SKS 레스토랑 (신규)
    ├── 호스트 관리 (다이닝/카페 2개 호스트)
    ├── 컨텐츠 관리
    └── 예약 관리
```

## 5. 운영 관리

### 5.1 좌석 현황

<Card title="실시간 좌석 현황" icon="th">
  **SKS 레스토랑 좌석 구성**
  - SKS 다이닝
    - 룸: 1개 (최대 8인)
    - 홀: 8개 테이블
      - 4인 테이블: H4-1 ~ H4-6 (6개)
      - 6인 테이블: H6-1 ~ H6-2 (2개)
  - SKS 카페
    - 별도 좌석 구성 설정

  **상태 표시**
  - 예약 가능 (초록색)
  - 예약됨 (노란색)
  - 이용 중 (빨간색)
  - 정비 중 (회색)
</Card>

### 5.2 체크인/쿠폰 처리

<Card title="운영 처리" icon="check-circle">
  **체크인 처리**
  - QR 코드 스캔
  - 수동 체크인
  - 노쇼 타이머 설정

  **쿠폰 처리**
  - SKS 다이닝/카페 쿠폰만 조회
  - 쿠폰 상세 확인
  - 사용 처리 (현장 결제 시)
  - 할인 적용
    - 식사권: 50%/30%/20%/10% 등급별
    - 음료권: 블랙 등급만
  - 취소 시 쿠폰 사용되지 않음
</Card>

### 5.3 실시간 알림

<Warning>
  **즉시 알림 항목**
  - 신규 예약 접수
  - 예약 취소 요청
  - 빈자리 알림 신청 (하루 10명 제한)
  - 노쇼 예정 (방문 시간 초과)
  - 시스템 오류
</Warning>

### 5.4 예약 오픈 관리

<Card title="예약 오픈 설정" icon="calendar-plus">
  **오픈 시점**
  - 매월 1일 오전 10시
  - 스케줄러 자동 실행
  
  **등급별 예약 가능 기간**
  - 블랙 등급: 익월 + 익익월
  - 나머지 등급: 익월만
  
  **알림 발송**
  - 예약 오픈 알림 신청자에게 푸시 발송
  - 알림톡은 발송하지 않음
</Card>
